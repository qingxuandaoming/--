## 🎯 项目简介

**灵境行者**（全称：非遗骑行导览系统）是一款面向骑行爱好者的智能路线规划与装备推荐平台。系统整合了高德地图 API 进行路径规划，通过 Python 爬虫采集电商平台骑行装备数据，为用户提供一站式骑行服务。

### 项目特点

- **智能路线规划**：基于高德地图 API，支持步行、骑行、驾车、公交多种交通方式
- **装备推荐与比价**：爬取京东、天猫等平台数据，提供装备搜索、价格监控、智能推荐
- **双后端架构**：Java 后端负责业务逻辑与数据管理，Python 后端负责爬虫与数据分析
- **响应式前端**：基于 Vue3 + Element Plus，适配桌面与移动端

---

## ⭐ 核心功能

### 1. 路线规划
- 起点/终点地址解析与坐标转换
- 多交通方式路径规划（步行、骑行、驾车、公交）
- 路线缓存与历史记录
- 路线收藏与评分

### 2. 装备服务
- 装备分类浏览与搜索
- 多平台价格对比
- 价格变动监控与预警
- 智能装备推荐

### 3. 用户系统
- 用户注册与 JWT 认证
- 个人骑行记录管理
- 装备收藏与价格提醒
- 设备绑定与数据同步

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0.0
- Java >= 17
- Python >= 3.8
- MySQL >= 8.0

### 一键启动（开发环境）

```powershell
# Windows
# 1. 启动 MySQL 服务
Start-Service MySQL80

# 2. 启动 Java 后端
cd app/java-backend
mvn spring-boot:run

# 3. 启动 Python 后端
cd app/python-backend
python app.py

# 4. 启动前端
cd app
npm run dev
```

详细部署步骤请参阅 [部署指南](../05-部署运维/01-部署指南.md)。
