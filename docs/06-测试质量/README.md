## 🧪 测试体系概览

### 测试金字塔

```
    E2E Tests (10%)
   ┌─────────────────┐
  │  Integration    │ (20%)
 │     Tests        │
└───────────────────┘
│   Unit Tests     │ (70%)
└─────────────────────┘
```

### 各端测试框架

| 端 | 单元测试 | 集成测试 | E2E 测试 |
|---|---------|---------|---------|
| 前端 (Vue3) | Vitest | - | Cypress / Playwright |
| Java 后端 | JUnit 5 + Mockito | Spring Boot Test | - |
| Python 后端 | pytest | - | - |

## 📈 主要改进成果

根据 [测试报告](./01-系统测试与改进报告.md) 记录：

| 改进项 | 改进前 | 改进后 |
|--------|--------|--------|
| 爬虫服务启动成功率 | 不稳定 | ~100% |
| 前端搜索响应时间 | ~1.5s | < 300ms |
| 问题定位时间 | 数小时 | 数十分钟 |
| 装备服务测试通过率 | 较低 | 明显提升 |
