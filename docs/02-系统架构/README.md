## 📊 数据库 ER 图速览

主要实体关系：
- `users` — 用户表（核心）
- `cycling_routes` — 骑行路线表
- `cycling_records` — 骑行记录表
- `equipment` / `equipment_categories` — 装备与分类
- `route_favorites` / `route_ratings` — 路线收藏与评分
- `crawler_tasks` — 爬虫任务表
